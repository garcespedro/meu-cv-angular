# 💼 Personal CV – Angular Web Application

---

# 🇬🇧 English Version

## 📌 Overview
This repository contains my personal **Curriculum Vitae (CV) Web Application**, developed using Angular.

The main goal of this project is to present my academic background, technical skills, and professional experience in a modern, responsive, and interactive web format. This application serves both as an online CV and as a demonstration of my front-end development skills.

---

## 🎯 Project Objectives
- Develop a responsive Single Page Application (SPA)
- Apply Angular’s component-based architecture
- Showcase professional and academic experience
- Demonstrate clean UI/UX principles
- Ensure scalability and maintainability

---

## 🏗️ Application Structure
The project follows Angular best practices and a modular structure:

- **Component-Based Architecture** – Reusable and organized components  
- **Responsive Design** – Optimized for desktop, tablet, and mobile devices  
- **Clean Code Organization** – Structured and easy to maintain  

---

## 📁 Folder Structure
- **src/**
  - **app/** – Main application components (About, Experience, Skills, Projects, Contact)
  - **assets/** – Static resources (images, icons, CV PDF)
  - **environments/** – Development and production configurations
- **angular.json** – Angular configuration file
- **package.json** – Project dependencies and scripts
- **tsconfig.json** – TypeScript configuration

---

## 🛠️ Technologies Used

### 💻 Languages
- **HTML5** – Application structure  
- **CSS3 / SCSS** – Styling and layout  
- **TypeScript** – Main development language  
- **JavaScript** – Core web logic  

### 🧰 Frameworks & Tools
- **Angular** – Front-end framework (SPA architecture)
- **Node.js** – Runtime environment
- **npm** – Package manager
- **Git & GitHub** – Version control and repository hosting

## 🚀 Features
